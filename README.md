## 📚 项目简介

班级管理系统是一款专为班主任设计的智能管理工具，集成了学生签到、随机点名、座位安排、随机抽奖和倒计时等六大核心功能，帮助教师轻松管理班级事务，提升教学效率和课堂互动性。

## ✨ 核心功能

### 1. 欢迎页面
- 系统介绍和功能概览
- 直观的用户界面引导

### 2. 学生签到
- 支持导入学生名单（Excel/CSV格式）
- 签到状态实时管理和监控
- 签到记录导出功能
- 出勤率统计和展示

### 3. 随机点名
- 炫酷的全屏旋转点名效果
- 可调节旋转时长
- 随机公平选择学生

### 4. 座位安排
- 灵活的课堂座位布局管理
- 支持拖拽调整座位
- 随机排座功能
- 座位布局保存和加载

### 5. 随机抽奖
- 随机抽取一名或多名学生
- 支持筛选未签到学生
- 适用于课堂奖励、提问等场景

### 6. 简易倒计时
- 自定义重要日期倒计时
- 实时显示剩余时间
- 多倒计时管理

## 🛠️ 技术栈

- **前端框架**: HTML5 + CSS3 + JavaScript
- **UI框架**: Tailwind CSS v3
- **图标库**: Font Awesome
- **图表库**: Chart.js
- **存储方案**: 浏览器本地存储 (localStorage)
- **响应式设计**: 适配桌面端和移动端

## 🚀 快速开始

### 安装部署

1. **克隆项目**
   ```bash
   git clone https://github.com/your-username/class-management-system.git
   cd class-management-system
   ```

2. **启动项目**
   - 直接在浏览器中打开 `index.html` 文件即可使用
   - 推荐使用 VS Code 的 Live Server 插件启动本地服务器
   
   ```bash
   # 使用 VS Code Live Server 启动
   # 或者使用 Python 简易服务器
   python -m http.server 8000
   ```

### 使用指南

1. **导入学生名单**
   - 点击"上传名单"按钮或拖放Excel/CSV文件
   - 系统支持标准的学号和姓名格式
   - 可下载模板文件参考格式

2. **开始签到**
   - 选择日期，点击"开始签到"
   - 系统会自动记录签到状态和时间
   - 可随时导出签到记录

3. **随机点名**
   - 调整旋转时长（1-10秒）
   - 点击"开始点名"按钮
   - 全屏显示旋转效果，点击"停止点名"查看结果

4. **座位管理**
   - 系统自动生成座位布局
   - 支持拖拽调整座位顺序
   - 可随机排座并保存布局

5. **随机抽奖**
   - 选择抽取人数（1名或多名）
   - 可选择仅从未签到学生中抽取
   - 点击"开始抽奖"查看结果

6. **设置倒计时**
   - 输入倒计时标题和目标日期
   - 点击"设置倒计时"
   - 系统会实时显示剩余时间

## 📊 数据管理

- 所有数据均存储在浏览器本地存储中
- 支持数据的导入、导出和清空
- 关闭浏览器后数据不会丢失

## 🔧 开发说明

```
class-management-system/
├── index.html       # 主页面
```

### 自定义配置

- **颜色主题**: 在 `tailwind.config` 中修改 `theme.extend.colors`
- **字体配置**: 在 `tailwind.config` 中修改 `theme.extend.fontFamily`
- **动画效果**: 在 `tailwind.config` 中修改 `theme.extend.animation`
